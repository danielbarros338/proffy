<strong>Proffy</strong> é uma plataforma para aproximar professores que desejam oferecer aulas, podendo registar sua disponibilidade
e custo, e alunos, que desejam encontrar o professor ideal para suas necessidades.

A plataforma oferece uma versão mobile adaptável a todo tipo de tela e uma versão desktop.

 - Projeto realizado na <strong>Next Level Week #2</strong>  da [Rocketseat](https://rocketseat.com.br/)
